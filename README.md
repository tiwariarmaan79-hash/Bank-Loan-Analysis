# 🏦 Bank Loan Analysis | End-to-End Data Analytics Project

An End-to-End Data Analytics project that analyzes bank loan performance using **Python, MySQL, and Power BI**. The project transforms raw banking data into meaningful business insights through data cleaning, SQL analysis, and interactive dashboards.

---

# 📖 Project Overview

Financial institutions process thousands of loan applications every month. Understanding loan performance, repayment behavior, borrower characteristics, and funding trends is essential for making informed lending decisions.

This project demonstrates the complete data analytics workflow:

- Data Cleaning using Python
- Business Analysis using MySQL
- Interactive Dashboard Development using Power BI

The final dashboard enables users to monitor loan performance, evaluate good vs bad loans, analyze borrower behavior, and explore lending trends using interactive visualizations.

---

# 🎯 Business Objectives

- Analyze overall loan performance.
- Monitor loan funding and repayments.
- Identify Good Loans and Bad Loans.
- Track monthly loan application trends.
- Analyze borrower demographics.
- Study state-wise lending performance.
- Understand loan purposes.
- Analyze home ownership and employee experience.
- Build an executive dashboard for business users.

# 📈 Key Insights

Some important findings from the analysis include:

- Good Loans account for approximately **86%** of all loan applications.
- Charged-Off loans represent around **14%** of applications.
- Debt Consolidation is the most common loan purpose.
- California has the highest funded loan amount.
- Fully Paid loans generate the highest repayment amount.
- Employee experience influences loan application distribution.
- Home ownership impacts borrowing behavior.
